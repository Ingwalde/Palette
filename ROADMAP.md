# Palette Roadmap

Status: completed.

- ES Modules
- Separate components, utilities and page logic
- Favorites with localStorage
- Export page
- Contrast checker

## v3.0 — Full-Stack Backend API

Status: completed.

- FastAPI backend
- SQLite database
- REST API for palettes

- Admin page for CRUD testing
- Admin token protection for create/update/delete actions
- PNG export with image preview
- Custom dropdown UI
- API documentation
